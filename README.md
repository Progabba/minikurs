# ЭОМ QUEST v2 · 8-битный курс «Альфа Инженер»

Ретро-игра в стиле Денди (NES): 7 миров по проектированию ЭОМ.
Псевдо-3D карта-мир, пиксельные canvas-анимации в каждом уроке,
боссы с HP-баром (Расчёт-Голем, Эксперт-Дракон), чиптюн-звук,
CRT-развёртка, тряска экрана, пиксельные взрывы. Без зависимостей.

## Что нового в v2
- Кабельные марки строго по ТИПУ ЗДАНИЯ (ГОСТ 31565 табл.2):
  жильё/офис -> нг(А)-LS; массовое пребывание/вокзалы/метро -> нг(А)-HF;
  детсад/школа/больница -> индекс LTx; пож.системы -> FRLS/FRHF (СП 6.13130)
- Пиксельные анимации по каждой теме (canvas)
- Боссы с полосой здоровья, ударами, взрывами и тряской экрана
- Анимированная 3D-карта: изо-блоки уровней, облака, шагающий герой

## Структура
- index.html
- assets/css/style.css — стили, CRT, боссы, тряска
- assets/js/chiptune.js — звук (Web Audio, без файлов)
- assets/js/data.js — 7 миров, теория, нормы. Ссылка на курс = COURSE_URL
- assets/js/anim.js — пиксельные canvas-анимации тем
- assets/js/games.js — мини-игры + бои с боссами
- assets/js/worldmap.js — анимированная карта-мир (canvas)
- assets/js/app.js — движок: старт, карта, уровни, HUD, финал, прогресс

## Публикация на GitHub Pages
Залей содержимое папки dendy в репозиторий -> Settings -> Pages ->
Deploy from a branch -> main -> /(root).

## Настройка
- Ссылка на курс — COURSE_URL в начале assets/js/data.js
- Прогресс/монеты — localStorage (ключ alfa_dendy_v1)
- Звук включается по PRESS START (требование браузеров)
